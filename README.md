# Stefan.dev Portfolio

A professional React portfolio website built with Vite.

## Project Structure

```
src/
├── components/
│   ├── Header.jsx
│   ├── Hero.jsx
│   ├── SocialLinks.jsx
│   ├── TechStack.jsx
│   └── FloatingSubscribe.jsx
├── styles/
│   ├── Header.css
│   ├── Hero.css
│   ├── SocialLinks.css
│   ├── TechStack.css
│   ├── FloatingSubscribe.css
│   └── App.css
├── App.jsx
└── index.css
```

## Features

- Clean, modern design
- Responsive layout
- Professional component structure
- Organized CSS in separate styles folder
- Tech stack showcase
- Social media links
- Floating subscribe element

## Development

```bash
npm run dev
```

## Customization

Update personal information in the component files:
- Hero.jsx: Name, title, description, location
- SocialLinks.jsx: Social media URLs
- TechStack.jsx: Technologies and skills
- Header.jsx: Navigation and branding

Styles are organized in the `src/styles/` folder for easy maintenance.
